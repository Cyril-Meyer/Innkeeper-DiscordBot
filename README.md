# Innkeeper-DiscordBot
Greeting and Chat Discord Bot

### Usage

**setup venv and install dependency**
```
python -m venv venv
# windows
.\venv\Scripts\activate.bat
pip install -U pip
pip install discord.py==1.7.3
```
